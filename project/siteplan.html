Color Schema

The company brand color is yellow, black, cream and green.
The primary color is yellow (#f7dc67). It will be used for headings, accents and highlight.
The secondary color is green (#8bc34a). It will be used for background colors, buttons and icons.
Text color is black (#000000).
It will be used for all paragraphs, captions, and other body text.
Background/neutral color is cream (#f5f5dc).
It will be used for page background and section background.

Typography

I will be using two font family of Open Sans and Montserrat.
The Open Sans will be used for the body text, button text and other UI element.
Montserrat will be used for the heading, title and acccent text.
